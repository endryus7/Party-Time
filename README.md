# 🎉 Party Time

Aplicação web desenvolvida em React para criação e gerenciamento de festas.  
O usuário pode cadastrar eventos e visualizar as informações de cada festa de forma simples e organizada.

## 🚀 Tecnologias

- React
- JavaScript
- HTML5
- CSS3
